# SynapseAI - AI Teaching Assistant

Веб-приложение для преподавателей с AI-функциями проверки заданий, генерации фидбека и анализа пробелов в знаниях студентов.

## Технологии

- **Next.js 14** (App Router)
- **TypeScript**
- **TailwindCSS**
- **shadcn/ui** компоненты
- **Zustand** для глобального состояния
- **Framer Motion** для анимаций
- **Recharts** для графиков
- **React Hook Form + Zod** для форм
- **Sonner** для уведомлений

## Структура проекта

```
synapseAI/
├── app/                    # Next.js App Router
│   ├── (marketing)/        # Маркетинговая страница
│   ├── auth/               # Страницы авторизации
│   └── app/                # Основное приложение (protected)
│       ├── dashboard/      # Главная страница
│       ├── courses/        # Курсы
│       ├── assignments/    # Задания
│       ├── insights/       # Аналитика
│       ├── assistant/      # AI ассистент
│       └── settings/       # Настройки
├── components/             # React компоненты
│   ├── ui/                 # Базовые UI компоненты (shadcn/ui)
│   ├── brand-logo.tsx      # Логотип бренда
│   ├── sidebar.tsx         # Боковая панель
│   └── topbar.tsx          # Верхняя панель
├── lib/                    # Утилиты и логика
│   ├── api.ts              # Mock API функции
│   ├── mock-data.ts        # Моковые данные
│   ├── types.ts            # TypeScript типы
│   └── utils.ts            # Утилиты
└── store/                  # Zustand store
    └── use-store.ts         # Глобальное состояние
```

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите dev сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте браузер:**
   ```
   http://localhost:3000
   ```

### Основные функции

- **Dashboard**: Обзор курсов, заданий и статистики
- **Курсы**: Управление курсами, просмотр студентов и заданий
- **Задания**: Список всех заданий с фильтрацией и поиском
- **Проверка заданий**: Интерфейс проверки работ студентов с AI-генерацией фидбека
- **Аналитика**: Графики освоения тем, студенты в зоне риска, частые пробелы
- **AI Ассистент**: Чат-бот для помощи в проверке и создании заданий
- **Настройки**: Профиль, тема, уведомления

## Логотип

Логотип должен быть размещен в `/public/logo.png`. В компоненте `BrandLogo` есть placeholder, который можно заменить на реальное изображение.

## Подключение Backend

Все API функции находятся в `lib/api.ts`. Замените моковые функции на реальные API вызовы:

```typescript
// Пример замены
export async function getCourses(): Promise<Course[]> {
  const response = await fetch('/api/courses');
  return response.json();
}
```
